# Calculadora

## Projeto

Desenvolvido durante o curso Back-End da Academia do Programador [Academida do Programador](https://academidadoprogramador.net) 2026

## Introduçao

Uma calculador de console simples mas poderosa, desenvolvida na linguagem c#. Ela permite realizar quato operações matemáticas e visualizar o histórico de operaçoes

## Funcionalidades

- **Operações Básicas**: Realize somas, subtrações, multiplicações e divisões com facilidade.
- **Tabuada**: A calculadora é capaz de gerar a tabuada de um número informado até o 10 (que será o multiplicador).
- **Histórico de Operações**: A calculadora é capaz de armazenar as operações anteriores.

## Como utilizar o programa

1. Clone ou baixe os arquivos do repositório.
2. Abra o emulador de terminal de preferência e navegue até a pasta raiz do projeto baixado.
3. Utilize o comando para restaurar as dependências do projeto.

``

    dotnet restore
``
4. Em seguida compile e execute o projeto com o comando:
``

dotnet run --project Calculadora.ConsoleApp
``

## Resquisitos

- .NET 10.0 SDK
